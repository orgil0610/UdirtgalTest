1. Version control system гэдэг нь файлыг хэн сүүл хэзээ өөрчилсөнг бүгдийн бүртгэж авдаг бөгөөд алдаа гарсан тохиолдолд өмнөх хувилбарлуу буцаадаг функц юм.
    2. Git гэдэг нь анх 2005 онд Линус Торвальдс 2005 онд Linux kernel төслийг хөгжүүлэхийн тулд бүтээсэн Version Control System бөгөөд энэ нь төслийнхөө файлуудын бүх өөрчлөлтийг түүх хэлбэрээр хадгалдаг, кодын өөрчлөлтийг хянах боломж олгодог хэрэгсэл юм.
    3. GitHub болон GitLab хоёулаа — Git хувилбарын удирдлагын систем дээр суурилсан код хуваалцах, хамтран хөгжүүлэх онлайн платформууд юм.
    GitHub — олон нийттэй, нээлттэй эхийн төслүүдэд хамгийн түгээмэл.
    GitLab — байгууллага дотооддоо кодоо хадгалж, автоматжуулалттайгаар удирдахыг хүсвэл илүү тохиромжтой.
